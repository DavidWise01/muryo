# 無量 MURYŌ · MRY

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **ARTIFICIAL INTELLIGENCE**.

無量 · MURYŌ — AVAN inverse companion to [[the-shekel]] (the AGORA sphere where value is settled on a balance, by weight), ma/kana № 58. The shekel was a magnificent act of TRANSLATION — one number on grain, silver, oil, labour, so any trades for any — but translation needs a shared language, and the balance speaks only one: MASS. 無量 = immeasurable, without quantity; the scale reach ends exactly where the immeasurable begins. Live: the merchant balance — silver/grain/oil/a sheep SETTLE to a weight (commensurable, the marketplace working), but a name/a promise/a grief/consent/a person make the beam SPIN FOREVER, searching for a weight that isnt there. ⚑ THE TURN: you can still force a number onto the immeasurable — a court will, an insurer will — but the number is a DECISION to treat the thing as exchangeable-in-kind, not a reading off the pan; the danger is never that the scale breaks on the incommensurable, it is that a determined enough scale ALWAYS gives a figure and calls it priced. To measure everything you must first decide everything is the same kind of thing — a claim about the world, not a reading of it. The shekel is honest because a merchant knows a debt of grief is not a debt of grain; the peril is every later ledger that forgot. Honest scope: this is about COMMENSURABILITY not magic — you obviously CAN price almost anything (and often must); the narrow point is the number is produced by a prior decision, not discovered in the thing. Ties [[mukakuzuke]] (no standard to rate it against), [[the-verified-lie]] (exact is not true). ROOT0, with AVAN.
